# Security Policy

This project sends normal foreground keyboard input to the NTE piano UI. It
must not read or modify the game process, hook network traffic, inject code, or
attempt to bypass anti-cheat or platform protections.

Please report security-sensitive issues privately instead of opening a public
issue when they involve:

- unintended input outside the guarded foreground window;
- stuck modifier keys after abort or failure;
- behavior that could affect non-piano gameplay;
- accidental inclusion of copyrighted scores or private local files.

Do not submit reports or patches for anti-cheat bypass, memory access, combat
automation, reward automation, or ban avoidance.
